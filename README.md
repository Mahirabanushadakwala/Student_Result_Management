# Student Result Management System

A simple Python project that manages student records using file handling.

## Features

- Add Student
- View Student Records
- Update Student Marks
- Calculate Result
- Grade Calculation
- Store Data in Text File

## Technologies Used

- Python
- File Handling

## Project Structure

Student_Result_Management/
│
├── main.py
├── students.txt
└── README.md

## How to Run

1. Install Python
2. Open the project folder
3. Run:

```bash
python main.py
```
